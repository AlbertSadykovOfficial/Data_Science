Убедитесь, что данные загруены корректно

Подробности:

Выглядит так, что вы открыли файл index.html напрямую, кликнув 2 раза на него(без сервера)

Так программа работать не будет, требуется запустить простой HTTP-сервер в каталоге, где располагаются файлы.

Откройте каталог с приложением и выполните команду:

Python 2: python -m SimpleHTTPServer

Python 3: python -m http.server 8000
